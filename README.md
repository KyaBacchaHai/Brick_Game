# Brick_Game

## General Information
You have 3 lives.
S: Shrink Paddle
X: Expand Paddle
E: Exploding brick
1/2/3: strength of brick at that position
N: unbreakable brick

## Instructions to run
Enter the command:
''' python3 main.py''

## Controls
a: move paddle to left
d: move paddle to right
p: release ball
q: quit game
